# Capacity Pressure Test v5 — 实验数据分析报告

**配置**: Config B (V=256, D=64, V/D=4)  
**架构**: Encoder-HoloTransformer(×3)-Decoder  
**任务**: Ring Span Inpainting (BERT-style masking, p=0.15)  
**训练**: 15000 epochs, lr=0.001, batch=32, warmup=500

---

## 一、核心结果总览

| 指标 | v4 (best) | v5 (best/final) | 变化 |
|------|-----------|-----------------|------|
| Train Accuracy | 93.88% | **100.00%** | +6.1% |
| Test Accuracy | 90.27% ± 3.0% | **99.60% ± 0.53%** | +9.3%, 方差↓5.7× |
| 温度 τ | 10.79 | **0.206** | ↓52× |
| p_target | 0.947 | **0.959** | +1.2% |
| neg% | 1.08% | **0.0%** | 完全消除 |
| Energy Gap | — | **1.435** | — |
| Ring NN% (L2) | 43.75% | **26.6%** | ↓17.2% |
| 训练时间 | 132s | 416s | ×3.2 |

**结论**: v5 在准确率上实现了质的飞跃——训练集完美拟合、测试集近乎完美（99.6%），且方差极小。但代价是环拓扑保持能力的大幅退化。

## 二、训练动力学分析

### 2.1 Loss 收敛特征

训练 loss（Boltzmann-Elegant）从初始的 ~6.09 下降至最终的 ~0.034（best epoch 14400），呈现三个清晰的阶段：

- **Phase 1 — 随机探索期 (epoch 0–2000)**：loss 从 6.09 缓慢下降至 ~3.24，train_acc 从 0% 攀升至 ~30%。模型此时在高维复数空间中进行初始的拓扑排列。τ 在此阶段从 0.12 上升至 ~0.28，反映竞争者能量分散度在增大——embedding 尚未形成清晰的分离结构。
  
- **Phase 2 — 快速学习期 (epoch 2000–8000)**：loss 加速下降至 ~0.2，train_acc 跃升至 ~99%。这是核心学习阶段，p_target 从 ~0.07 陡增至 ~0.86，neg% 从 ~58% 降至接近 0%。rho（学习进度指标）在此阶段从 0 开始加速增长至 ~0.43。

- **Phase 3 — 精细调整期 (epoch 8000–15000)**：loss 在 0.03–0.18 之间小幅波动，train_acc 稳定在 100%。τ 平稳收敛至 ~0.21，p_target 稳定在 0.93–0.97。rho 继续缓慢上升至 0.78，表明 embedding 空间仍在持续优化。

### 2.2 泛化能力

test_acc 的采样较稀疏（每 1000 epoch 一次），但趋势清晰：1000→0.13, 3000→0.60, 5000→0.91, 7000→0.97, 10000→1.00, 14000→1.00。训练-测试 gap 始终极小（最终 ~0.4%），表明模型没有过拟合——v5 的 Encoder-Decoder 架构和 BERT masking 提供了强正则化效果。

### 2.3 Embedding RMS 演化

EmbRMS 从初始的 1.0 持续下降至最终的 0.804。这个下降趋势延续了 v4 的行为（v4 降至 0.68），但 v5 的下降幅度更小、速度更慢。这意味着 v5 的 Encoder-Decoder 架构部分缓解了 embedding 模长萎缩问题——Decoder 的可学习投影提供了额外的尺度自由度，使 embedding 本身不需要承担所有的距离编码压力。

## 三、Boltzmann 热力学诊断

### 3.1 温度 τ 的稳定性

v5 的 τ 最终收敛至 ~0.206，相比 v4 的 ~10.8 低了 **52 倍**。这是一个根本性的改变：

- v4 中 τ ~10 意味着 softmax 输入 \(-E_k / \tau\) 的有效范围很窄（能量差异被 τ 稀释），分类决策"模糊"。
- v5 中 τ ~0.2 意味着即使很小的能量差异也能产生极大的概率差，分类决策"锐利"。

τ 大幅降低的原因：Encoder-Decoder 架构将 Transformer 输出映射回 Embedding 空间时，Decoder 的权重尺度控制了 \(\tilde{E}_k\) 的绝对大小。当 Decoder 学到恰当的尺度后，竞争者能量的标准差（即 τ）自然收敛到一个小值。这本质上等价于在 logit 层添加了一个可学习的温度缩放。

### 3.2 p_target 与 neg%

p_target 最终达到 0.959（最佳 0.969），远高于随机基准 1/V = 0.0039。neg%（target 能量高于最近竞争者能量的比例）在 epoch ~8600 后稳定为 0%，表明所有 256 个 token 的 embedding 都已被推到各自的能量最低点。

### 3.3 Energy Gap

归一化 Energy Gap（\(\tilde{E}_{nw} - \tilde{E}_{tgt}\)）从初始的 ~0.5 持续增长至 1.43。从 Energy Landscape 图可见：target 能量集中在极低值（~0.02–0.05），而最近竞争者的能量分布在 1.3–2.0 之间，中位数 ~1.64。没有任何 token 出现负 gap，说明所有 token 都被正确分离。

Self-Boltzmann p_target 的分布（图 07d）显示均值为 0.663，这个指标是纯粹基于 Embedding 几何计算的静态分类概率（不经过 Transformer），表明 Embedding 本身已经编码了大量的辨别信息。

## 四、嵌入空间几何分析

### 4.1 PCA 投影

PCA 前三个主成分分别解释 2.7%、2.6%、2.4% 的方差。方差解释率极低且高度均匀，这是一个关键信号：

- 256 个 token 的 embedding 在 64 维复数空间（128 维实数空间）中近乎**各向同性**地展开。
- 没有明显的低维流形结构——特别是，**没有出现期望的环形结构**。
- PCA 2D 投影中，不同 Token ID 的颜色呈随机分布，没有渐变过渡。

这与 v4 的观察一致：排斥力机制使 embedding 趋向于高维散点编码（类似 random hash code），而非保拓扑的频率编码。

### 4.2 距离矩阵

全局距离矩阵（图 06）呈现高度均匀的绿色调，L2 距离分布近似正态，中位数 9.17，min 6.43，max 11.32。关键观察：

- 距离矩阵中没有显著的对角线带状结构（相邻 token 距离不显著小于远处 token），确认了环拓扑的缺失。
- 局部放大（50×50）可见一些微弱的块状聚集，但不构成系统性的拓扑结构。
- Welch Lower Bound = 0.1085，JL Upper Bound = 0.2081，实际模长 RMS 0.77 落在合理的几何约束区间内。

### 4.3 模长分析

Per-token RMS 均值为 0.810 ± 0.044，分布集中但存在个别离群值（range ~0.68–0.92）。Per-dim RMS 均值为 0.802，但前几个维度（dim 0–5）的 RMS 显著高于后续维度（~0.95 vs ~0.80），说明低频维度承载了更多的辨别信息。模长热力图中可见 dim 0–3 有一些 token 模长特别大（>2.0），这些维度恰好对应相位线性度最高的维度。

## 五、相位结构与频率编码

### 5.1 相位线性度

64 个维度的相位线性度分数中：

- Dim 0: 0.275（最高）
- Dim 1: 0.205
- Dim 2: 0.167
- Dim 3–63: 本质为 0

没有任何维度超过 0.5 阈值（high_linearity_count = 0）。这意味着**没有维度呈现出与 Token ID 线性相关的相位进动**——即模型没有学到理论预期的"每个维度编码一个频率分量 \(\theta_d(k) = 2\pi f_d \cdot k / V\)"的结构。

### 5.2 Unwrapped Phase

Phase Unwrap 图（图 04d）中，Dim 0 的 unwrapped phase 从 0 上升至 ~55（约 8.8 个完整周期），Dim 1 约 2 个周期，但波形高度不规则，充满局部波动。这不是理想的线性进动，而是一种"趋势性但噪声很大"的相位结构。

### 5.3 频率覆盖

理论分析显示 \(\alpha = 1.99\)（接近 2.0 的理论极限），freq_cov = 64/128 = 50%。D=64 复数维度最多编码 64 个独立频率，而 V=256 的环的完美 DFT 编码需要 128 个频率。模型需要通过非线性组合弥补剩余 50% 的频率覆盖——从 99.6% 的测试准确率来看，3 层 Transformer 的非线性推理成功补偿了这个理论缺口。

## 六、环拓扑保持能力

### 6.1 NN% 分析

NN%（L2）= 26.6%（68/256），NN%（BE）= 20.3%（52/256）。这意味着仅有约 1/4 的 token 的最近邻是环上的真实邻居。相比 v4 的 43.75%，这是一个显著退化。

从 NN 分布图（图 05a）可见：大多数 token 的最近邻确实是环距 1 的邻居（绿色柱），但有大量 token 的最近邻环距达到 7–20（包括对径位置 d=20 的明显尖峰）。

### 6.2 Ring Distance vs Embedding Distance

关键图表 05b 揭示了核心问题：embedding 距离与 ring distance 几乎**完全不相关**。散点图呈现出一个近乎水平的带状分布——无论环距是 1 还是 128，embedding 距离都在 7–11 的狭窄区间内波动。这确认了模型采用了"高维散点编码"而非"保拓扑编码"。

### 6.3 邻居排序

Neighbor Rank 的中位数为 25。这意味着一个 token 的环上邻居在 embedding 距离排序中平均排在第 25 位——远好于随机的 128，但远差于理想的 1。

## 七、注意力模式分析

图 10 展示了 4 个注意力头在一个典型 masked 输入（MASK 位置 [15, 16, 17]）上的模长和相位模式。

### 模长模式
- **Head 0** 呈现稀疏的局部关注——mask 区域（row 15–17）主要关注邻近的未 mask 位置（14, 18 附近），同时有少量对序列首尾的关注。
- **Head 1** 更分散，关注较远的位置（特别是 key 25–30 区域），可能负责收集全局上下文。
- **Head 2** 模式与 Head 0 类似但更分散。
- **Head 3** 在 mask 行呈现对序列末端（key ~25–31）的集中关注。

### 相位模式
所有 4 个头的相位模式都呈现高度随机的"马赛克"纹理，没有可辨识的系统性结构（如对角线条纹或块状区域）。这与 embedding 缺乏相位结构的观察一致——注意力的相位传递主要是高维的"噪声式"编码。

## 八、v4 → v5 架构变更效果评估

v5 引入了四项关键变更，逐一评估其贡献：

### 8.1 Encoder-HoloTransformer-Decoder 架构
**效果：极为显著**。将 Embedding 空间与 Transformer 工作空间解耦后：
- Decoder 的可学习投影提供了隐式的 logit 温度缩放，使 τ 从 ~10.8 降至 ~0.21。
- Embedding 不再需要直接服务于 Transformer 的中间计算，可以更自由地编码 token 身份。
- EmbRMS 下降趋势减缓（0.80 vs v4 的 0.68），部分缓解了模长萎缩。

### 8.2 3 层 Transformer Block（vs 1 层）
**效果：关键性提升**。3 层允许分阶段推理：定位上下文 → 整合信息 → 输出准备。这使模型即使在 embedding 不保拓扑的情况下，也能通过多步非线性推理弥补频率覆盖的 50% 缺口，达到 99.6% 准确率。

### 8.3 BERT-style Masking
**效果：中等**。每步有效训练位置从 ~96 增至 ~154（+60%），显著提高了训练效率和数据利用率。但也可能削弱了 span inpainting 任务对环拓扑的依赖——随机 mask 不像连续 span 那样强制要求模型理解局部连续性。

### 8.4 Linear Warmup
**效果：辅助性**。500 epoch 的 warmup 帮助 Polar Adam 的二阶矩稳定估计，减少了早期训练的振荡。

## 九、关键发现与存在的问题

### 正面发现

1. **准确率质的飞跃**：从 v4 的 90.3% 到 v5 的 99.6%，且方差从 3.0% 降至 0.53%。
2. **Boltzmann 热力学完全健康**：τ 稳定、neg% = 0、energy gap 持续增长。
3. **完美的泛化**：train-test gap < 0.5%，无过拟合迹象。
4. **训练全程稳定**：best 与 final 模型指标完全一致，无退化。

### 存在的问题

1. **环拓扑完全丧失**：NN% 从 v4 的 43.8% 降至 26.6%，embedding 距离与 ring distance 不相关。模型选择了"高维散点编码"而非"保拓扑编码"来解决任务。
2. **相位结构缺失**：没有维度呈现有意义的频率编码。复数域的相位优势未被充分利用——模型更多地利用了高维空间的容量而非复数的代数结构。
3. **PCA 方差完全分散**：前 3 个 PC 仅解释 ~8% 方差，embedding 是高维均匀散布。
4. **训练时间增加 3.2×**：3 层 block + 更多 epochs 带来计算开销。

### 根因推测

v5 的 Encoder-Decoder 架构过于"灵活"——Decoder 可以学习任意的线性映射，使 Transformer 无需在 embedding 空间中保持拓扑结构就能高精度完成任务。这是一种"捷径学习"：模型找到了一种更容易优化的编码方案（高维散点），而非理论上更优雅的编码方案（保拓扑频率编码）。

## 十、后续实验建议

1. **拓扑正则化损失**：添加一个显式的辅助 loss 惩罚 embedding 距离与 ring distance 的 Spearman 相关性偏离，迫使模型保持拓扑结构。
2. **约束 Decoder 为正交/酉投影**：限制 Decoder 的权重矩阵为酉矩阵，使其只能做旋转和缩放而非任意线性变换，从而保持 Transformer 空间与 Embedding 空间的几何一致性。
3. **Ablation 实验**：单独测试"只加 Encoder-Decoder 不加 3 层"和"3 层但不加 Encoder-Decoder"，分离两个因素的贡献。
4. **回退 span masking 测试**：在 v5 架构上使用 span masking（`--mask-mode span`），观察是否能恢复更强的环拓扑归纳偏置。
5. **增加 V/D 压力**：在 V/D=8, 16 的更高压力配置下测试，观察 v5 架构的容量边界。
