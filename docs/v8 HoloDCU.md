## Anla v8 研究总结：从诊断到架构再到实验验证

---

### 一、问题诊断阶段

#### 1.1 v7 实验的三重结构性缺陷定位

对 v7 autograd 实验数据（复数 61.27% vs 实数 69.30%）的分析，定位了三个相互关联的底层缺陷：

**缺陷 1：输出层的信息投影损失。** `Re(ẑ^H ŵ_k)/τ` 的 cosine logits 丢弃了 `Im(z^H w)` 分量——等价于信号处理中的单路正交解调（homodyne detection），只测量波函数的一个正交分量。同时 cosine 归一化固定了 logits 值域为 $[-1/\tau, 1/\tau]$，阻止了模型通过 embedding 模长表达分类确信度。

**缺陷 2：全纯激活函数 $f(z) = z + \alpha z^2$ 的表达力不足。** Liouville 定理的直接推论——有界全纯函数不能产生稀疏激活（零点孤立定理），无法实现 GELU 的核心功能：选择性抑制不匹配的特征通道。Cauchy-Riemann 方程将自由度砍半（$\mathbb{C}^{64} \to \mathbb{C}^{64}$ 的全纯映射只有 $64^2$ 个独立导数，而 $\mathbb{R}^{128} \to \mathbb{R}^{128}$ 有 $128^2$）。

**缺陷 3：复数线性层的参数空间是实数线性层的真子群。** $GL(n, \mathbb{C}) \subset GL(2n, \mathbb{R})$，复数乘法的旋转-缩放约束对不具备旋转对称性的字节序列是错误的归纳偏置。

**统一诊断：** 三个缺陷归结为一个根本问题——Anla 的复数架构假定数据具有连续的旋转-相位对称性，但字节级 NLP 数据不具备。

#### 1.2 v7 的 AGComplexAttention 未利用全息特性

关键代码发现：v7 实验中的 attention 使用了 `Re(Q^H K)` + 实数 softmax——**在注意力分数阶段就丢弃了相位信息**。而项目中已有的 `HolographicAttention` + `MagPhaseSoftmax`（复数 softmax + 相位透传）从未在 NLP 实验中被测试过。这意味着 v7 的 61.27% 是在**双重信息瓶颈**（attention 的 Re 投影 + 输出的 Re 投影）下获得的。

### 二、物理学理论框架

#### 2.1 宏观-微观类比的三层结构

建立了 RVNN-CVNN 与经典-量子物理的系统性类比：

| 层面 | 物理学                     | 神经网络                       |
| -- | ----------------------- | -------------------------- |
| 描述 | 宏观用实数（位置、温度），微观用复数（波函数） | RVNN 处理宏观事实量，CVNN 处理相位结构数据 |
| 涌现 | 经典从量子通过退相干涌现            | 实数输出从复数内部表示通过学习型投影涌现       |
| 架构 | 有效场论在每个尺度用最自然的描述        | 混合架构在不同子系统用最匹配的数域          |

#### 2.2 关键物理学引用与映射

* **量子力学中复数的必要性之争（Renou 2021 → Hoffreumon 2025）：** 复数不是额外的魔法，而是紧凑的编码方式。实数理论可以通过维度加倍再现复数理论的所有预测——这直接映射到 v7 中实数模型用 1.9× 参数量超越复数模型的实验事实。

* **退相干理论与 `Re()` 投影的同构：** 建立了偏迹（partial trace）与 `Re()` 投影之间的精确数学对应——完整密度矩阵 → Hermitian 内积，偏迹 → Re 投影，丢失的非对角元素 → 丢失的 Im 分量。

* **有效场论（EFT）为混合架构提供理论基础：** 复数子系统 = UV（高能/微观）理论，实数子系统 = IR（低能/宏观）理论，ℂ→ℝ 过渡层 = RG 流中的匹配条件。

* **对称性破缺假说（Physical Review Research 2023, npj AI 2025）：** cosine 归一化将 embedding 锁定在单位超球面上，等价于在 Ising 模型中施加无穷强约束场——阻止了自发对称性破缺。

### 三、架构设计的关键突破

#### 3.1 DCU（Decoherence Coding Unit）的从零推导

**出发点：** 用户提出将 DCU 作为学习型 ℂ→ℝ 映射，基于零差检测原理。

**物理推导过程：**

1. 从平衡零差检测的光学设置出发：信号场 $a_s$ + 本振 $a_\text{LO} = |\alpha|e^{i\theta}$ → 分束器 → 两探测器 → 差分
2. 差分输出 $I_- = 2|\alpha| \cdot \text{Re}(a_s \cdot e^{-i\theta})$ —— 本振相位 $\theta$ 选择测量哪个正交分量
3. 映射到 DCU：可学习本振 $c_m = \alpha_m e^{i\theta_m}$，判据信号 $s_m = \text{Re}(c_m^* h_m)$

**QND（量子非破坏性测量）的引入：** 将复数信号的两个正交分量分配不同角色——$s_m = \text{Re}(c_m^* h_m)$ 作为判据（测量 $\hat{X}$），$t_m = \text{Im}(c_m^* h_m)/|c_m|$ 作为内容（未扰动的 $\hat{P}$）。这实现了判据信息和内容信息的**完美正交分离**，避免了自门控方案中判据对内容的「污染」。

**双路门控设计：**

* 判据路径：$o^{(1)} = \text{GELU}(s - b)$（自门控，保留幅度信息）
* 内容路径：$o^{(2)} = \sigma(s - b) \cdot t$（交叉门控，sigmoid 纯开关）
* GELU 用于判据路径（保留幅度），sigmoid 用于内容路径（不改变 t 的尺度）

#### 3.2 FFN 门控问题的深层分析——「复数域中符号消失了」

**核心洞察：** 实数 GELU 的门控建立在「符号」判据上（正=匹配，负=反匹配），但复数没有全序关系，没有符号概念。

**解决方案推导：** 在复数域中，等价于「符号」的概念是**信号与某个参考方向的对齐程度**：$\text{Re}(c_m^* z_m) = |c_m| \cdot |z_m| \cos(\arg(z_m) - \theta_m)$。这恢复了匹配（$\cos > 0$）vs 反匹配（$\cos < 0$）的二元判据。

**门控目标的精确回答：** 控的是整个复数信号 $z_m$（实部和虚部被同一个实数标量同步缩放）；判据来自信号在学习到的方向 $c_m$ 上的投影；这个投影是坐标系不变的。

#### 3.3 DCU 与 FFN 的统一——结构性架构简化

**用户关键洞察：** 复数域门控的 ℂ→ℝ 投影步骤和 DCU 在数学上是同一个操作——既然 FFN 的门控本身就需要一次退相干，为什么不在这一步直接完成退相干，让 FFN 在实数域中工作？

**架构重构：** DCU 不是网络末端的一次性操作，而是**每层 Transformer 内部的结构性组件**。

最终的单层结构：

```
x(ℂ) → RMSNorm → HoloAttn(纯复数) → +x(残差)
     → RMSNorm → W₁_ℂ(D→4D) → DCU(ℂ→ℝ: 零差检测+双路门控) → W₂_ℝ(8D→2D) → reshape(ℝ→ℂ) → +x(残差)
```

**物理读法：** 每层执行一次「量子干涉（HoloAttn）→ 退相干测量（DCU）→ 经典处理（GELU）→ 量子反馈（reshape 回 ℂ）」循环。复数残差贯穿始终。

#### 3.4 Weight Tying 的复数域自然形式

**Hermitian 内积方案：** $\text{logit}_v = \text{Re}(E[v]^H \cdot x) + b_v$

核心论证：这不是 v7 的 `Re()` 投影（那个有 cosine 归一化和固定值域），而是**无约束的 Hermitian 内积**——模长自由，logit 无界。在展开的实数表示下等价于标准 weight tying。物理含义：每个 token 的 embedding 作为自己的本振做零差检测。

#### 3.5 不需要 $W_2$ 偏置的数学-物理论证

从数学角度：常数偏移 $\Delta$ 的信息含量极低（不携带位置/上下文信息），大部分效应被后续 RMSNorm 吸收。从物理角度：$b_2$ 对应探测器的暗电流——系统误差，应被校准消除。从工程角度：破坏了「零门控→零修正」的恒等映射性质。

### 四、实验验证的关键结论

#### 4.1 性能指标——复数首次全面超越实数基线

| 指标             | HoloDCU (ℂ) | Real (ℝ) |
| -------------- | ----------- | -------- |
| Final Test Acc | **68.81%**  | 64.88%   |
| Final Test PPL | **2.865**   | 3.236    |
| 实数等效参数量        | **429K**    | 628K     |

复数模型用 68% 的参数量，取得了更高的准确率和更低的 PPL。参数利用率高出 55%。

#### 4.2 与 v7 的对比

v7 复数（autograd）：61.27% test acc → v8 HoloDCU：68.81% → **提升 7.5 个百分点**。v7 到 v8 的核心改动：MagPhaseSoftmax 保留相位 + DCU-FFN 的 ℂ→ℝ→ℂ 退相干循环 + weight tying 的 Hermitian 内积 + 放弃 cosine 归一化。

### 五、权重分析的物理发现

#### 5.1 本振相位保持完美均匀分布

三层 Rayleigh R < 0.05。网络没有学到偏好特定测量方向——信号在复平面上没有特权方向。验证了「逐通道独立 DCU 在 $W_1$ 已做全局混合前提下是足够的」这一设计判断。

#### 5.2 门控阈值实现了「渐进退相干」

Layer 0 的 b ≈ 0（无选择性，53.9% 正），Layer 1-2 的 b 正移（97-98% 为正）。物理对应：浅层做无偏信息提取，深层做选择性指针态筛选——量子达尔文主义的计算类比。

#### 5.3 Embedding 模长的「反常对称性破缺」

与理论预测**相反**：高频 token 学到更小的模长（1.46-1.63），低频 token 模长更大（2.05-2.22）。

解释：output bias 承担了频率先验编码（r = -0.730 强负相关），embedding 模长承担判别区分度——高频 token 需要精细方向分辨（小模长+高方向敏感性），低频 token 的模长膨胀是梯度噪声累积的结果。网络自组织形成了 embedding-bias 互补编码系统。

#### 5.4 层间参数的单调趋势

W1 行模长、RMSNorm scale、本振增益|c|、门控阈值 b 均逐层递增——信息从浅层到深层经历了「逐步放大和精化」的过程。

### 六、方法论层面的贡献

1. **从量子测量理论推导神经网络组件设计：** DCU 不是凭直觉拼凑的，而是从零差检测的物理原理逐步推导出每个计算步骤的数学形式和参数含义。
2. **用 EFT 的思想指导混合架构设计：** 不同的计算子系统使用最匹配其处理对象对称性的数域——这不是工程妥协，而是物理学的第一性原理。
3. **「渐进退相干」作为多层复数网络的组织原则：** 每层内部的 ℂ→ℝ→ℂ 循环不是信息损失，而是有目的的信息整理——等价于物理退相干中指针态的逐步涌现。
4. **权重分析作为物理理论的实验验证：** 门控阈值的层间正移、本振相位的均匀性、embedding-bias 互补编码——这些训练后的参数分布不仅验证了设计的正确性，还揭示了网络自组织出的、超出预期的计算策略。
